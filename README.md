项目简介
本项目基于RK3566 嵌入式 Linux平台，面向无人机机载场景，实现机载 H.264 视频 UDP 分片传输、上位机实时解码预览、视频本地持久化录制、串口飞控指令转发整套嵌入式业务链路。

项目架构
机载端（RK3566）：
    视频数据接收 → UDP 分片封装 → 网络上行发送；
    监听上位机指令 → 串口解析转发至飞控；
上位机地面站：
    UDP 接收视频流 → FFmpeg 解码渲染 → 实时画面预览；
    一键录制保存 MP4；下发控制指令；
    指令链路：地面站 UDP 指令 → RK3566 解析 → UART 串口下发 → 飞控执行返航等动作。
